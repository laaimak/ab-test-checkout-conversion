# A/B-тест: редизайн страницы оформления заказа

Учебный / pet-проект: полный цикл статистического анализа A/B-теста —
от формулировки гипотезы до бизнес-рекомендации.

## Задача

Проверить гипотезу: упрощённый дизайн страницы checkout повышает конверсию
в покупку по сравнению с текущей версией сайта.

- **H0:** конверсия в группах не отличается
- **H1:** конверсия в тестовой группе выше

## Данные

Синтетические данные эксперимента (`data/ab_test_checkout.csv`) — ~10 350
пользователей, 2 группы (control / treatment), 14 дней эксперимента.
Эффект и шум заложены намеренно для демонстрации корректного статистического
анализа (реалистичный, не «идеальный» результат).

## Что сделано

- Sanity-check рандомизации (проверка баланса групп по устройствам)
- Точечная оценка конверсии по группам
- **Z-тест для двух пропорций** (проверка значимости различия)
- **Доверительные интервалы** (метод Уилсона) для визуальной проверки значимости
- Проверка на **novelty effect** — динамика конверсии по дням эксперимента
- Разбивка эффекта по устройствам (гетерогенность)
- Анализ вторичной метрики — выручки на пользователя (t-test)
- Итоговые бизнес-рекомендации

## Результаты

| Группа | Конверсия | 95% ДИ |
|---|---|---|
| Control | ~10.5% | — |
| Treatment | ~12.6% | — |

**p-value < 0.05** — различие статистически значимо, относительный прирост ~+20%.

**Рекомендация:** раскатать новый дизайн на 100% трафика; отдельно проверить
эффект на мобильных пользователях в follow-up тесте.
